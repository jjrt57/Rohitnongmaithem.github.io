## Featured Work
* **[QE_2_CIF](https://github.com/jjrt57/QE_2_CIF):** An open-source Python script that parses final relaxed atomic coordinates from raw Quantum ESPRESSO output logs and converts them to standard `.cif` format.This removes a manual bottleneck, enabling direct import into visualisation software like VESTA and Mercury.
* **[The Quantum ESPRESSO Necronomicon](https://medium.com/@rohitnongmaithem):** A systematic debugging series published on Medium explaining root causes of cryptic HPC errors (like OOM memory kills, `IEEE_INVALID_FLAG` exit codes, and node migration data corruption) for a technical computational audience.
* **Computational Software:** Quantum ESPRESSO, VESTA, BURAI, pymatgen, PySCF, GROMACS, MERCURY, XcrySDen
* **Programming Languages:** Python (data parsing, automation), JS, SQL, Swift.
* **Core Techniques:** DFT/ab initio calculations, HPC batch jobs, FRET analysis, UV-Vis Spectrophotometry, Photoluminescence (PL).
